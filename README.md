# 🤖 College Help Desk Chatbot  
### ⚡ Developed by Nexus Neuron

---

## 📌 Project Overview
The College Help Desk Chatbot is designed to assist students by answering common queries related to college information such as fees, admissions, facilities, and academic details.

This chatbot uses a **JSON-based knowledge system** to provide fast and accurate responses without requiring internet-based AI APIs.

---

## 🎯 Features
- 💬 Interactive chatbot UI
- 📚 Predefined college FAQ system
- ⚡ Instant responses
- 🎨 Modern purple-blue-pink themed interface
- 🤖 User-friendly chat experience
- 🔍 Keyword-based intelligent matching

---

- Python 🐍
- Flask 🌐
- HTML, CSS, JavaScript 🎨
